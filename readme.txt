== Description == 

Do you have issues creating your desired big, creative menu? We are here to help! Mega Max Menu is a great tool to create any menu that you have in mind.

Create small or big, horizontal or vertical menus, by using your imagination. Add images to your menu and customize font styles. Get mobile friendly, responsive menu for your website.

Customize submenus add custom CSS and JS codes. Enter widget areas to auto-generate the contents from your widgets.

To recreate the same menu on other websites, you can try our Import and export option. To explore more options, install the plugin and give it a try!
